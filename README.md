# Timer
# ⏱️ Timer em JavaScript

Projeto de um **timer (temporizador)** desenvolvido com o objetivo de praticar lógica de programação, manipulação do DOM e controle de tempo utilizando JavaScript.

---

## 🚀 Funcionalidades

* Definir tempo em **horas, minutos e segundos**
* Iniciar o timer
* Pausar o tempo (Stop)
* Continuar de onde parou
* Resetar o timer
* Exibição do tempo em formato **HH:MM**

---

## 🧠 Conceitos praticados

* Manipulação do DOM (`getElementById`)
* Eventos com `onclick`
* Uso de `setInterval` e `clearInterval`
* Controle de estado com variáveis globais
* Conversão de tempo (horas → segundos)
* Formatação de dados (`padStart`)

---

## 💻 Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco em:

* Melhorar a lógica de programação
* Entender o funcionamento de timers
* Criar uma base sólida para projetos mais complexos
